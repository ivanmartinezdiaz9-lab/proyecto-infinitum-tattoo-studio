# 🖤 Infinitum Tattoo Studio

Sitio web oficial de **Infinitum Tattoo Studio**, un estudio enfocado en la creación de piezas únicas con identidad, técnica y criterio artístico.

---

##  Concepto

Este proyecto no busca ser solo una web informativa, sino una **experiencia visual** que represente el proceso artístico del tatuaje:

* Minimalismo oscuro
* Estética editorial
* Narrativa visual
* Enfoque en identidad

---

##  Tecnologías utilizadas

* HTML5
* CSS3 (custom + responsive design)
* JavaScript (interacciones y animaciones)
* Git & GitHub

---

##  Características

*  Diseño responsive (desktop + mobile optimizado)
*  Animaciones suaves y performance-friendly
*  Dirección artística personalizada
*  Timeline narrativa del estudio
*  Sección de equipo con enfoque premium
*  Integración directa con WhatsApp

---

##  Objetivo del proyecto

Crear una presencia digital que refleje:

> “Cada tatuaje es una obra única, y cada cliente una historia distinta.”

---

🟢 En desarrollo activo — mejoras continuas en diseño, performance y experiencia.
